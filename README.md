# Customer Behavior Analysis

## Project Overview

This project analyzes customer purchasing behavior using data analysis and visualization techniques. The goal is to identify customer segments, understand purchasing patterns, and detect potential churn risks to help businesses improve customer engagement and retention.

The analysis was performed using Python in Jupyter Notebook with libraries such as Pandas, NumPy, Matplotlib, and Seaborn.

---

## Objectives

* Analyze customer transaction data.
* Identify purchasing patterns and trends.
* Perform customer segmentation using clustering.
* Detect potential customer churn behavior.
* Provide actionable business recommendations.

---

## Dataset

The dataset contains customer transaction and demographic information such as:

* Customer ID
* Customer Name
* Age
* Gender
* Purchase Date
* Product Category
* Product Price
* Quantity
* Total Purchase Amount
* Payment Method
* Returns
* Churn

## Project Workflow

### 1. Data Loading

The dataset was imported into Jupyter Notebook using Pandas.

### 2. Data Cleaning

* Removed missing values
* Converted purchase date into datetime format
* Created new features such as Month and Repeat_Customer

### 3. Exploratory Data Analysis (EDA)

Data was explored to understand distributions, correlations, and purchasing patterns.

### 4. Data Visualization

Several visualizations were created, including:

* Purchase amount distribution
* Product category analysis
* Gender-based purchase behavior
* Monthly sales trends
* Customer segmentation visualization
* Correlation heatmap

### 5. Customer Segmentation

K-Means clustering was used to group customers based on purchasing behavior.

### 6. Insights & Recommendations

Key insights were extracted to help businesses improve marketing strategies and customer retention.

---

## Key Insights

* Customers aged 25–40 contribute significantly to total purchases.
* Certain product categories generate higher sales than others.
* High-value customer clusters were identified through segmentation.
* Repeat customers play an important role in revenue generation.
* Monthly trends reveal seasonal purchasing behavior.

---

## Business Recommendations

1. Introduce loyalty programs to retain high-value customers.
2. Provide targeted promotions for churn-risk customers.
3. Focus marketing efforts on popular product categories.
4. Encourage repeat purchases through rewards and discounts.
5. Use customer segmentation for personalized marketing strategies.
